## コミットメッセージPrefix一覧

コミットメッセージは`[prefix] メッセージ`の形式で記載する。

| Prefix    | 用途                                               | 例                                                 |
|-----------|----------------------------------------------------|----------------------------------------------------|
| [feat]    | ユーザーに価値を生む新機能追加                     | [feat] 旅行プランのキャンセル機能を追加           |
| [fix]     | バグ修正                                           | [fix] 重複予約が作成される不具合を修正             |
| [perf]    | パフォーマンス改善                                 | [perf] 行き先検索結果をキャッシュ                  |
| [refactor]| 挙動を変えずに内部構造を整理                     | [refactor] 旅程マッパーをモジュールに分割          |
| [style]   | 挙動に影響しないフォーマット・スタイル調整         | [style] GraphQLスキーマのフォーマットを統一        |
| [docs]    | ドキュメントの追加・更新                           | [docs] 予約APIの利用手順を更新                    |
| [test]    | テストの追加・修正                                 | [test] TravelPlanリポジトリのエッジケースを追加    |
| [chore]   | ビルドツールや依存更新など雑務（挙動変更なし）     | [chore] NestJSを最新マイナーにアップデート         |
| [build]   | ビルド設定・依存に関わる変更（実行物へ影響するもの）| [build] webpackのバンドル解析を追加                |
| [ci]      | CIの設定・スクリプト変更                           | [ci] シード付きE2Eジョブを追加                     |
| [revert]  | 過去コミットの取り消し                             | [revert] \"[feat] 旅行プランのキャンセル機能を追加\" |
